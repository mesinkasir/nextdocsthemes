## Tema

Program restoran ini sudah modern dan dukungan tema keren bisa kamu pilih via fitur skin, pilih tema sesuai dengan warna selera kamu disini.

---

## Screen

Pilih metode screen display penjualan, kami saran kan menggunakan maximal agar tampilan kasir semakin nyaman untuk digunakan.


---

## Mode POS

Selanjutnya pastikan menggunakan transaksi penjualan langsung atau pencatatan meja, jika penjualan langsung pilih tiket mode standard , untuk pencatatan meja pilih mode restaurant.


---

## Cashregsiter

kita perlu untuk mengatur peraturan printer kasir disini agar nantinya kita mudah dalam mencetak struk kasir, maupun mencetak laporan print out via berbagai dukungan hardware device disini. pastikan printer kasir sudah terinstall untuk memudahkan dalam prsoes integrasi dengan printer kasir, selanjutnya pilih printer kasir yang digunakan.

---

## Locale

Untuk locale pastikan memilih semua system by default, atau gunakan anydesk agar team kami dapat membantu melakukan setting secara tepat untuk program restoran software resto cafe ini.
