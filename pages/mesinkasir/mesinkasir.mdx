# Mesin Kasir

Untuk mendukung kebutuhan penjualan dan pembukuan secara stabil dan cepat maka system real point of sale benar benar dibutuhkan untuk menunjang kebutuhan itu, maka berikut adalah paket set mesin kasir siap pakai untuk resto cafe rumah makan kuliner.Menggunakan all in one touchscreen by ASUS dengan display monitor touchscreen dan pc cpu desktop menjadi satu menyempurnakan kebutuhan POS system.
